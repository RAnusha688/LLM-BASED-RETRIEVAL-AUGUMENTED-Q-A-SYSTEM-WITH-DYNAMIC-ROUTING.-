## 🧠 About the Project

This project is a Retrieval-Augmented Generation (RAG) AI agent that answers user questions by combining LLM capabilities with document context. It retrieves relevant information from uploaded documents and generates accurate, context-aware responses. The system is designed to demonstrate practical GenAI integration, backend API development, and Docker-based deployment.
---

## 📂 Dataset

This project does not rely on a traditional dataset.

Users can upload documents (PDFs, text files, or CSVs) as context for the AI agent.

Embeddings are generated from uploaded documents to allow semantic search for relevant information.

---

## 🧱 Technologies Used

- Programming Language: Python 3.11
- Frameworks: FastAPI, LLM APIs, Vector embeddings
- Deployment & Packaging: Docker, Uvicorn
- Generative AI: OpenAI API (LLMs), RAG techniques for context-based answering

## 🧠 Model Architecture

1.Document Embedding Generation:
Documents are converted into vector embeddings using a pre-trained SentenceTransformer model.

2.Vector Search:
Queries are matched against document embeddings using FAISS to find the most relevant chunks.

3.LLM Question Answering:
The retrieved chunks are passed to an LLM (via API) to generate precise and contextually accurate answers.

4.API Layer:
FastAPI serves as the backend, allowing users to send queries and receive answers in real-time.

## ⚙️ Installation

1.Clone the repository
git clone https://github.com/yourusername/LLM-BASED-RETRIEVAL-AUGUMENTED-Q-A-SYSTEM-WITH-DYNAMIC-ROUTING.git

2.Create and activate a virtual environment:
  python -m venv venv
  source venv/bin/activate   # Linux/macOS
  venv\Scripts\activate      # Windows

3.Install required dependencies
  pip install -r requirements.txt
  
4.Create and activate a virtual environment:
  python -m venv venv
  source venv/bin/activate   # Linux/macOS
  venv\Scripts\activate      # Windows

5.Set up environment variables in .env:
  OPENAI_API_KEY=your_openai_api_key

6.Build and run Docker container:
  docker build -t ai-agent-app .
  docker run -p 8000:8000 --env-file .env ai-agent-app
  
▶ Usage

1.Start FastAPI server locally:
  uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

2.Send a POST request to /query with:
{
  "question": "Explain the RAG system?",
  "documents": ["doc1.pdf", "doc2.txt"]
}

Receive a context-aware answer generated by the AI agent.
